== Description ==
WPDB-Optimize resizes your Wordpress DB indexed columns to more appropriate sizing for faster lookups and database performance.

The default database columns sizing provided by Wordpress is desiged for flexibility, not performance.


== Installation ==

1. Upload the plugin files to the /wp-content/plugins/wpdb-optimize directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->WPDB Optimize screen to configure the plugin
4. Use Tools->WPDB Optimize to run the optimization process.
